About

TweetShare is an Istanbul Bilgi University Comp 461 (Web Development) Project. It is a project for testing our web development knowledges with Django(which is becoming popular web freamework) and using twitter API.

It's basically a portal for sharing user tweets and gives a chance to make comment(s) to friends tweet(s). You can even post inside updates(in future, I'm planning to add making post to twitter with authentication to Twitter.).
